# treesize-cli

A simple and efficient command-line interface (CLI) tool to analyze and visualize directory sizes and disk space usage.

> **Note:** This project was created for the purpose of learning and practicing modern C++ development. It is an educational project to explore language concepts, file management, and the use of third-party libraries.

## Features

- Recursive directory analysis.
- Display of file and directory sizes.

## Tech Stack

- C++ compiler compatible with C++17 or higher.
- [CMake](https://cmake.org/) (for building).
- [Conan](https://conan.io/) (for dependency management).



## Usage

Once compiled, you can run the executable from the `bin` folder (or the output location):

```bash
./treesize-cli [options] [directory_path]
```

Example:
```bash
./treesize-cli C:\MyDirectory
```
